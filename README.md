# Trust Audit Platform — Liechtenstein

Professionelle Audit-Software für einen Liechtensteiner Trustee mit ~2'000 Trust-Mandaten.

## Demo

Klickbarer Prototyp mit:
- Dashboard mit Heatmap und Mandatsübersicht
- Trustee Control Center (IKS-Prüfung mit dokumentierten Stichproben)
- Portfolio Testing (Populationen, Sampling-Engine)
- Exceptions Center
- Mandatsakte mit Annahmeerklärung, Vollständigkeitserklärung, Prüfungs-Workbench, Bericht
- Audit Trail

## Setup

```bash
npm install
npm run dev
```

Öffne [http://localhost:3000](http://localhost:3000).

## Deploy auf Vercel

1. Push dieses Repository auf GitHub
2. Gehe zu [vercel.com](https://vercel.com) → New Project → GitHub Repo auswählen
3. Vercel erkennt Next.js automatisch → Deploy klicken
4. Fertig — Live-URL wird generiert
